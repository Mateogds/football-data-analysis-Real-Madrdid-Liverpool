# Análisis de datos de fútbol: Real Madrid vs Liverpool (UCL Final 2017/18)

## Descripción
Este proyecto analiza la final de la UEFA Champions League 2017/18 entre Real Madrid y Liverpool, utilizando datos abiertos de StatsBomb.

## Análisis realizados
- Red de pases del Liverpool
- Mapa de pases de Toni Kroos

## Herramientas utilizadas
- Python (pandas, mplsoccer)
- StatsBomb Open Data

## Principales insights
- Liverpool presenta una alta concentración de conexiones en banda derecha
- Toni Kroos actúa como eje organizador del juego, con alta participación en zona central y capacidad para cambiar la orientación del juego
